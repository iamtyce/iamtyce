So what exactly is a UI developer?

Glad you asked. It means I specialise purely in SASS / CSS3, HTML5 and responsive web design. I don't work in backend, dynamic code or even in Javascript.

Why? Specialists make the world beautiful. They care about the little things and work with other specialists to create truly incredible products.
